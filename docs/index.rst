Welcome to edge-gwas Documentation
===================================

**edge-gwas** is a Python package for performing Genome-Wide Association Studies (GWAS) 
with edge computing capabilities, optimized for large-scale genomic data analysis.

Introduction
------------

edge-gwas provides efficient tools for:

* Large-scale GWAS analysis
* Edge computing optimization for genomic data
* Integration with modern data platforms
* Scalable statistical analysis
* Visualization and reporting

Key Features
------------

- Two-stage analysis: calculate alpha on training data, apply to test data
- Detects under-recessive, recessive, additive, dominant, and over-dominant effects on a continuous scale
- Handles binary and quantitative outcomes
- Support for PLINK format data
- Built-in visualization functions
